LÄS DEL 1

HTML

TAGS

Tags, eller en tag, är en del av språket som indikerar början och slutet av ett HTML-element. HTML-taggen <html> specificerar roten ur ett HTML dokument.

head

Head är huvuddelen av en hemsida och innehåller meta-datan av hemsidan. DVS. en mängd innehåll som inte visas på hemsidan utan fungerar i bakgrunden.

meta charset

Meta charset attributet definierar hemsidans teckenkodning. Numera så brukas oftast UTF-8 vilket definieras som följande: <meta charset="utf-8">

style

Style taggen definierar ett CSS-block där du kan ange CSS-attribut för dina HTML-element. Definierars som följande: <style></style> <style src=””></style>

body

Body taggen definierar kroppen av ditt HTML dokument där du anger allt som sedan visas på din webbläsare. Anges som: <body></body>

span

Span taggen är en inline-tag som inte har någon riktig betydelse utan brukas mest inuti andra taggar för enkel separering och formatering.

div

En div definierar en sektion eller en behållare för andra element. Brukas mest för att separera delar av ditt dokument med andra delar.

EXTRA TAGS

table En “table” eller “tabell” innefattar celler, rader, samt kolumner och brukas för att bilda tabeller. h1-6 H, eller Heading Tag, anger en titel. p P, eller Paragraph, anger paragrafer i html. br BR, eller break, anger linjebryte. Används sällan efter större utveckling av CSS. a a, eller hyperlink, anger länkar. img img, eller image, anger bilder. ul, ol, li ul, unordered list, ol, ordered list, li, list-item section Alternativ till div och anges för att definiera en sektion av din hemsida.

CSS

HUR MAN ANVÄNDER CSS

Det finns flera sätt att bruka CSS, eller Cascading Style Sheet, men det allra bästa sätt är att skriva din css i en extern css fil. <link rel="stylesheet" href="style.css">

SKILLNADER MELLAN HTML OCH CSS

CSS används för att bygga up utseendet på ditt HTML-dokument med hjälp av HTML-elementens attributer.

border

“Border” skapar en ram runt de element du anger. Border har följande egenskaper:border-color, border-style, border width.

color

“Color” anger färgen på en given text i format HEX, RGB, RGBA, HSL, HSLA, Color Keywords.

alt-attributet

ALT-attributet till bilder (img) anger en alternativ text för screen-readers eller om bilden saknas.

VIKTIGA CSS ENHETER

ABSOLUTE UNITS px, pt, in, cm, mm. RELATIVE UNITS em, rem, %, ex, ch, fr. VIEWPORT UNITS VW, VH, vmin, vmax

RESPONSIV DESIGN

Responsiv design handlar om att se till att din hemsida fungerar för olika skärmstorlekar. Det kan definieras som följande. <meta name="viewport" content="width=device-width, initial-scale=1.0">

POSITIONERING I CSS

Hur man positionerar element i CSS är väldigt debatterat och bör brukas olika i specifika fall.

display: flex;

Flex, eller flexbox, positionering är baserat på elementets “children” alltså elementet inuti de element du sätter flexbox på. Flexbox är alltid relativt till föräldern.

POSITION: RELATIVE;

Relativ position är baserat på avståndet från rotpositionen av elementet.

POSITION: ABSOLUTE

Absolut position är det absoluta positionen av förälderns storlek.

POSITION: FIXED

Definierar en fixerad position baserat på webbläsarens storlek.

MARGIN

Margin utgör avståndet utanför givet element och kan brukas för att bilda avstånd mellan element.

PADDING

Padding utgör avståndet innanför givet element och kan brukas för att bilda avstånd inuti ett element.

Mjukvarupatent

Mjukvarupatent (även programvarupatent och IT-patent) är patent som omfattar datorprogram, algoritmer, affärsmetoder och liknande.[källa behövs] Foundation of a Free Information Infrastructure (FFII) föreslår att programvarupatent bör definieras som "patent på något som utförs av en dator med hjälp av ett datorprogram".[1] Programvarupatent i betydelsen IT-implementerade affärsmetoder är tillåtet i bland annat USA, men formellt inte tillåtet i Europa, men andra former av programvarupatent beviljas regelmässigt av europeiska patentverket EPO med stöd av traditionell patentlagstifning.[förtydliga] Ett omdebatterat direktivförslag i EU föreslog [när?] att programvarupatent även i betydelsen affärsmetoder skulle tillåtas, men förkastades av EU-parlamentet.
